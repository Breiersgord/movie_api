this API will return a list of ALL movies to a user and consider their search criteria in returns. genres users will be able to explore include thriller, psychological, supernatural, slasher, fantasy, comedy, splatter, B, monster, paranormal, and survival (to name a few). a user can look up a specific movie by title and return all of the information about that specific movie, as well as the director information associated with it. in order to save their searches / create favorites / etc., users will also have the ability to register with userinfo (and subsequently unregister, if desired.)
